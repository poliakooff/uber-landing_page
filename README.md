**Использованные технологии:**

1. HTML: Использование языка разметки для создания структуры и содержания страницы.
2. CSS: Применение каскадных стилей для визуального оформления лендинга.
3. JavaScript: Добавление интерактивных элементов и динамического поведения на странице.
4. Git: Контроль версий для управления и отслеживания изменений в коде проекта.
5. GitHub: Хостинг репозитория для сохранения и обмена кодом с другими разработчиками.

**Описание лендинга:**

Лендинг представляет собой одностраничный веб-сайт, посвященный компании "Uber Partners". Основной целью лендинга является привлечение водителей, которые хотели бы сотрудничать с компанией "Uber" и зарабатывать на своих автомобилях. На лендинге представлена информация о преимуществах работы с компанией "Uber Partners", а также указаны условия и требования для водителей, как с собственными автомобилями, так и с автомобилями компании.

Лендинг имеет простой и интуитивно понятный дизайн. Основное внимание уделено текстовой информации и визуальным элементам, которые помогают передать ключевые преимущества работы с компанией "Uber Partners". Также на лендинге присутствует контактная информация, позволяющая потенциальным водителям связаться с компанией для получения дополнительной информации или оставить заявку.

В целом, этот лендинг служит простым и практическим примером веб-разработки, демонстрируя применение различных технологий для создания привлекательного и функционального одностраничного сайта.
